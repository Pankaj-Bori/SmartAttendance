# User Stories

## Parent
- As a parent, I want to receive a notification when my child boards the bus.
- As a parent, I want to see my child’s attendance history.

## Admin
- As an admin, I want to view real-time bus location on a map.
- As an admin, I want to see the list of students who attended each day.
- As an admin, I want to send notifications to parents.

## Device Operator
- As a bus staff, I want the attendance reader to work automatically without manual input.
