# Project Requirements

## Functional Requirements
- The system must capture attendance automatically using RFID, QR codes, or mobile app.
- Real-time location tracking of school buses via GPS.
- Send push and SMS notifications to parents regarding attendance and bus status.
- Admin dashboard to manage attendance, view bus routes, and handle notifications.
- Secure login for parents and admins.

## Non-functional Requirements
- System must be reliable and available during school hours.
- Real-time updates with minimal delay.
- Data privacy must be maintained securely.
- Scalable architecture to handle multiple schools.
